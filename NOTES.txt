Types of CSS in Web Development

1. Inline CSS

Inline CSS is applied directly to a specific HTML element using the
style attribute.Usage: Best for quick, one-off style changes or testing
specific elements.

Where and How to use: It's used directly within the opening tag of element using style attribute

---------------------------------------------------------------------------

2. Internal CSS

Internal CSS is defined within a "style" tag in the head section of an
keep styles separate from content

Where and How to use: It's used inside head element within style element

---------------------------------------------------------------------------

3. External CSS
External CSS is defined in a separate .css file and linked to the HTML
document using the "<link />" tag.Usage: Ideal for styling multiple pages
consistently and maintaining a clean separation between content and
presentation.

Where and How to use: We need to create separate file and link it with html using <link> tag / element

The CSS Box Model is a fundamental concept in web design that treats every HTML element as a rectangular box

Layers of the Box Model
Content :text,image
Padding: space between Content and Border
Border: A line that is around padding
Margin: space after Border
